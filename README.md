# Beduketal_2026_ESR_Homeownershipmobility
Code repository for replicating 

Bedük, S. Benassi, E. & Lersch, P. (forthcoming). Are the children of homeowners increasingly advantaged? Trends in relative and absolute mobility of homeownership in Europe. European Sociological Review 

The analysis uses data from EU-SILC 2011 and 2019 including register, household and individual files and intergenerational module variables.

Eurostat. (2024). European Union Statistics on Income and Living Conditions: Cross-sectional UDB 2004–2023, 2024 release 1 [Data set]. European Commission. https://doi.org/10.2907/EUSILC2004-2023

Data access conditions: Eurostat grants access to microdata for scientific purposes in accordance with Regulation (EU) No 557/2013 on access to confidential data for scientific purposes. The access is restricted to universities, research institutes, national statistical institutes, central banks inside the EU, as well as to the European Central Bank. Individuals cannot be granted direct data access.

See for an overview of data: http://ec.europa.eu/eurostat/web/microdata/overview

See for how to apply for data access: 
http://ec.europa.eu/eurostat/documents/203647/203698/How_to_apply_for_microdata_access.pdf/82d98876-75e5-49f3-950a-d56cec15b896

To run the code, you need to change the directories to specify the location of inputs and outputs in each do file. The outputs will then be saved to the specified working directory.

To replicate the whole analysis, you can run the Master Do file (DECIPHE_P3_DX_master.do) (after changing relevant directory names within each do file including the master), which will call each do file and execute them in order.

To run the analysis, we used STATA 18.
